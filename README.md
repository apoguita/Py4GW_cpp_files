# Py4GW C++ Source


# This repository is retired

The Py4GW C++ backend has been rewritten and now lives at:

    https://github.com/apoguita/Py4GW_Reforged_Native

Please use the new repository for everything going forward:

  - Latest source code and updates
  - Build instructions and releases
  - Issues, questions, and pull requests

The code in this repository is frozen at its final state. It is kept online
only as a historical archive and will no longer receive updates, fixes, or
support. Its role has been fully replaced by Py4GW_Reforged_Native.

Thank you to everyone who used and contributed to the original backend.
